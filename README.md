* **Objective:** Design a functional 20'×16' residential living room layout including optimized spacing, door placement, and five windows according to specification.
* **Tools Used:** AutoCAD 2D Drafting, Dynamic Blocks, Annotative Scaling.
* **Key Deliverables:** 
  * `Mehreen Irfan_2D_FLOOR PLAN.dwg` (CAD Drawing File)
  * `preview.png` (Layout Preview)
